About wpwp project
Well Played Web Play
This project is a step for being interactive developer.
I really inspired from Jogmin Kim and Studiomeal(1분 코딩).

I used css 3D skills to make my webpage 3D which react from user's scroll. And most of this project's content is made with html canvas with js that makes content more beautiful and interactive. Also, there are two projects that I made in army where I have few leisure time, these are also make me take a step for being interactive developer.
And I thought it was all playing for me to care user intearction and make canvas animation, and even the project that I made in army. So I decided to name this project to 'Wpwp' which means 'Well played web play'.
